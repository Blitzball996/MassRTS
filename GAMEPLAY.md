# MassRTS 游玩指南

## 游戏概述

MassRTS 是一款大规模实时战略游戏，灵感来自《史诗战争模拟器》。通过 GPU 计算着色器驱动战斗和移动 AI，支持最高 **100万** 单位同屏对战。你将指挥红色军团，对抗蓝色 AI 敌军，争夺地图上的战略据点。

---

## 操作指南

### 相机控制
| 操作 | 按键 |
|------|------|
| 缩放 | 鼠标滚轮 |
| 平移 | WASD / 方向键 |
| 旋转 | 鼠标中键拖动 |

### 部队控制
| 操作 | 按键 |
|------|------|
| 框选部队 | 左键拖动 |
| 移动选中部队 | 右键点击地面 |
| 设置集结点 | 中键点击（新兵在此生成） |
| 核弹瞄准 | N键进入 → 左键发射 |
| 调整购买数量 | +/- 键（每次±500） |

### 商店（购买兵种）
- **左侧面板点击购买**：直接用鼠标点击商店按钮即可购买兵种
- **数字键 1-9**：快捷键购买（同时保留）
- 新兵会在 **集结点**（绿色旗帜标记）处生成
- 中键点击地面可随时更改集结点位置

## 经济系统

- **被动收入**：每秒自动获得金币
- **击杀赏金**：消灭敌军获得额外金币
- **据点加成**：占领据点增加每秒收入
- **核弹解锁**：积累足够分数后可发射核弹（大范围毁灭）

---

## 胜利条件

地图上分布 **5个战略据点**：

| 据点 | 位置 | 初始控制 |
|------|------|----------|
| 中央高地 | 地图正中 | 中立 |
| 西部前哨 | 左侧 | 红方(己方) |
| 东部前哨 | 右侧 | 蓝方(敌方) |
| 北方山口 | 上方 | 中立 |
| 南方渡口 | 下方 | 中立 |

### 占领规则
- 派遣部队进入据点范围（圆形区域）
- 区域内只有己方单位时开始占领
- 双方同时存在 = **争夺中**（进度暂停）
- 占领进度满后据点归属改变

### 获胜方式
1. **据点胜利**：控制 3个以上据点并持续 30秒
2. **歼灭胜利**：消灭所有敌方单位

## 地图介绍

### 1. 平原战场
开阔平原，少量丘陵。地形平坦，骑兵和大规模冲锋的天堂。

### 2. 河谷争夺
中央大河分隔两军。需要抢占渡河点或绕行，弓箭手可隔河射击。

### 3. 山地要塞
群山环绕，仅有数条通道。防御方优势巨大，火炮和投弹手是破局关键。

### 4. 沙漠风暴
一望无际沙漠，视野开阔，无处躲藏。纯粹的硬碰硬战斗。

### 5. 密林伏击
茂密森林覆盖大部分区域。移动速度降低，适合弓箭手和伏击战术。

### 6. 堑壕绞肉
一战风格堑壕遍布战场。投弹手和火炮为王，正面冲锋代价惨重。

## 战术建议

### 经典阵型
```
        [ 火炮 火炮 ]          ← 后排远程
      [ 弓箭 弓箭 弓箭 ]      ← 中排输出
    [ 盾兵 步兵 步兵 盾兵 ]    ← 前排肉盾
         [ 骑兵 骑兵 ]         ← 侧翼等待时机
```

### 战术要点
- **骑兵侧翼**：不要正面冲锋，绕后切弓箭手
- **盾兵前排**：200HP坦克保护脆皮远程
- **火炮消耗**：200码射程安全输出，但需要前排保护
- **投弹手破阵**：对密集阵型伤害爆炸，用来破盾墙
- **地形利用**：上坡减速、下坡加速、沼泽大幅减速、河流几乎无法通行
- **集结点**：设在安全位置，避免新兵被敌军屠杀

### 经济博弈
- 前期用民兵拖延时间积攒经济
- 中期核心兵种成型开始争夺据点
- 后期火炮+核弹远程消耗

## 性能说明

- **GPU 计算着色器**：移动AI、战斗AI、空间哈希全部在 GPU 上运行
- **实例化渲染**：所有单位通过硬件实例化一次性绘制
- **LOD系统**：远处单位自动切换为广告牌（Billboard）
- **视锥剔除**：屏幕外单位不渲染
